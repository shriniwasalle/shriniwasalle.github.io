# Shriniwas Alle Portfolio

This repository contains the personal portfolio website for Shriniwas Alle, built as a static HTML page.

## Contents

- `index.html` — the main portfolio page.

## About

The portfolio showcases Shriniwas Alle as a Senior QA Engineer II and Specialist Automation Engineer, highlighting:

- professional summary and technical strengths
- automation, DevOps, and quality engineering skills
- work experience and achievements
- certifications and education

## Usage

1. Open `index.html` in a web browser to view the site locally.
2. Optionally, host the folder on any static web host such as GitHub Pages, Netlify, or Vercel.

## Deployment

To publish this repository with GitHub Pages:

1. Push the repository to GitHub.
2. In the repository settings, enable GitHub Pages from the `main` branch.
3. Set the site folder to `/` if prompted.

## Contact

For updates or questions, edit the `index.html` file and customize the contact details directly in the page.
