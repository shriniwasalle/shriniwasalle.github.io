# Shriniwas Alle Portfolio

A premium static portfolio website built with HTML, CSS, and JavaScript. This site is designed for direct hosting on GitHub Pages, Netlify, or any static site platform.

## Files

- `index.html` — main portfolio page
- `style.css` — visual styling and responsive design
- `script.js` — animations, navbar behavior, theme toggle, and form interaction
- `assets/resume.txt` — downloadable resume companion file
- `assets/images/` — placeholder folder for additional portfolio images
- `assets/pdf/` — placeholder folder for resume or document uploads

## Local Preview

1. Open `index.html` in a modern browser.
2. The page includes a loading animation and smooth scrolling experience.
3. For the contact form, the submit action opens your default email client.

## Deployment

### GitHub Pages

1. Push the repository to GitHub.
2. Enable GitHub Pages in repository settings.
3. Select the `main` branch and root folder `/` as the publishing source.
4. Visit the generated GitHub Pages URL.

### Netlify

1. Connect the repository to Netlify.
2. Configure the build settings (no build command required).
3. Deploy as a static site.

## Customization

- Update personal details and project descriptions in `index.html`.
- Replace `assets/resume.txt` with a PDF file in `assets/pdf/` if desired.
- Adjust colors and fonts in `style.css`.
- Add portfolio screenshots to `assets/images/` and reference them from `index.html`.

## Notes

- No backend or database is required.
- This is a single-page responsive portfolio.
- Built with HTML5, CSS3, and vanilla JavaScript.
